Runs a model
Available features are buffer, intersect, difference, combine and union.
Need to select the model, based on selection possible layers will appear. After choosing the layer/layers user can hit a create button to get the model.
Distance for the buffer can be given by user otherwise model will create using default distance.
User guidelines to create JSON file :
1) Type of functionalities/operations must be a string's and always small letters with correct spelling eg. buffer, intersect...
2) The id must be an integer eg. 1, 2.
3) If the layer taking input using from the previous operation, thus the value of the layer should be integer of that operation
4)The operation must take input from preperformed or preprocessed operations
parameters :
1) Layer names must be string's
2) Buffer distance must be integer and can not be a negetive value
3) User definable parmeters must be boolean
4) Similarly for dissolve also
Json file example
        {	"model_name" : "my_model",
            "operations" : [
                            {
                             "type": "buffer",
                             "id"  : 1,
                             "parameters" :{
                                  "l1": "input_layer1",
                                  "radius": {
                                    "default_dist" : 50,
                                    "user_defineable" : false,
                                    "units" : "kilometres",
                                    "dissolve" : true
                                  }
                             }
                            }, 
                            {
                             "type": "difference",
                             "id"  : 2,
                             "parameters" : {
                              "l1": "input_layer3",
                              "l2": 1 
                             }
                            }
                        ]
        }
